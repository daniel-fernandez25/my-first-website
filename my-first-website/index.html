Cuatro reglas de diseño

Espaciado

Una regla simple que siempre se rompe. Debes juntar grupos de elementos similares y después ponerlos entre espacios vacíos. El espaciado puede ser una herramienta muy poderosa para cualquier desarrollador.

Cuadrícula

Las dimensiones de los elementos y las distancias entre ellos se organizan en una estructura cohesiva conocida como cuadrícula o "grid". Cuando un elemento está fuera de la cuadrícula, sabes que algo no está bien.

Fuentes

Para esta regla, no debe haber más de tres tipos de fuentes diferentes en un solo sitio web. Una debe ser para el encabezado, otra para el cuerpo y una tercera para resaltar palabras esenciales.

Colores

Además del color principal, también hay colores de acento que se usan para destacar los detalles importantes. Utiliza las herramientas adecuadas para ayudarte a seleccionar una buena combinación de colores.
